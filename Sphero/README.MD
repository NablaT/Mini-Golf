### SPHERO MINI-GOLF

## SETUP

- Go in the Sphero folder.
- Do `npm install`.

## START

- Go in the Sphero folder.
- Do `grunt` or `npm start`.

## TEST

- Go in the Sphero folder.
- Do `npm test`.

## ARCHITECTURE

- The `app.js` file contains all we need to creates a functional node js app.
- The `sockets` folder contains the socket communication with the sphero device.
