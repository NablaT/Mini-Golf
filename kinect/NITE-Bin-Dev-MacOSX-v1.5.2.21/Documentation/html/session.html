Session

A session is a state in which the user is in control of the system using hand points. While in session, hand points are tracked and have persistent IDs.

There are 3 possible states:

  • Not in session: Looking for a gesture
  • In session: Tracking hand points
  • Quick refocus (optional): Looking for the gesture and an additional quick refocus gesture.

The session is encapsulated in the XnVSessionManager object. Each hand point is represented by the XnVHandPointContext structure.
